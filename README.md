```
  # clone project
  git clone https://github.com/LikaiLee/wechat.git

  // 在 ./config/index.js 中配置微信相关信息 => TOKEN, APP_ID, APP_SECRET
  
  # install dependencies
  npm install

  # serve with hot reload at localhost:80
  npm run dev

  # build for production
  npm run prd

```
